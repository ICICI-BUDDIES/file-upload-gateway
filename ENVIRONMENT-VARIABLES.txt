ENVIRONMENT VARIABLES FOR DEPLOYMENT
===================================

SET THESE BEFORE STARTING THE JAR FILES:

SPRING_PROFILES_ACTIVE=prod
DB_HOST=<your-mssql-server-ip>
DB_PORT=1433
DB_NAME=templatedb
DB_USERNAME=template_user
DB_PASSWORD=<database-password>

EXAMPLE:
SPRING_PROFILES_ACTIVE=prod
DB_HOST=192.168.1.100
DB_PORT=1433
DB_NAME=templatedb
DB_USERNAME=template_user
DB_PASSWORD=MySecurePassword123!

HOW TO RUN:
1. Start template-service first: java -jar template-service-1.0.0.jar
2. Start gateway-service second: java -jar gateway-service-0.0.1-SNAPSHOT.jar

PORTS USED:
- Template Service: 8081
- Gateway Service: 8080
